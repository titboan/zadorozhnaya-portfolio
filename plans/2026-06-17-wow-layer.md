# План: «сайт, с которого не хочется уходить» — WOW-слой

Статус: завершён

## Контекст

MVP готов и задеплоен: https://zadorozhnaya-portfolio.vercel.app  
Стек: React + Vite + plain CSS, 4 секции, Intersection Observer анимации, фото из Figma.

Задача: добавить эффекты, которые превращают сайт из «хорошего» в «запоминающийся».  
Ориентир — тренды 2026: плавность, микро-взаимодействия, персонажность интерфейса.

---

## Уровень 1 — Максимальный эффект, минимум кода

- [x] **Lenis — плавный скролл** (`main.jsx` + убран `scroll-behavior: smooth` из `App.css`)
- [x] **Hover на карточках Works** — zoom `scale(1.05)` + тёмный overlay «СМОТРЕТЬ» (`Works.css`, `Works.jsx`)
- [x] **Grain/noise текстура** — `body::after` с SVG-шумом, pure CSS (`App.css`)

## Уровень 2 — Персонажность интерфейса

- [x] **Кастомный курсор** — белый круг 12px, на hover Works → 80px + «СМОТРЕТЬ» (`Cursor.jsx`, `Cursor.css`)
- [x] **Параллакс в Hero** — фото движется медленнее страницы (`Hero.jsx`)
- [x] **Счётчик «уже 7 лет»** — цифра отсчитывается от 0 при загрузке (`useCounter.js`, `Hero.jsx`)

## Уровень 3 — Редакционный стиль

- [x] **Анимация заголовка по словам** — каждое слово Hero появляется отдельно снизу вверх (CSS `wordSlide`, `App.css`, `Hero.jsx`)
- [x] **Marquee-строка** между Services и Works — бесконечная прокрутка тегов (`Marquee.jsx`, `Marquee.css`)
