# План: «сайт, с которого не хочется уходить» — WOW-слой

Статус: в работе

## Контекст

MVP готов и задеплоен: https://zadorozhnaya-portfolio.vercel.app  
Стек: React + Vite + plain CSS, 4 секции, Intersection Observer анимации, фото из Figma.

Задача: добавить эффекты, которые превращают сайт из «хорошего» в «запоминающийся».  
Ориентир — тренды 2026: плавность, микро-взаимодействия, персонажность интерфейса.

---

## Уровень 1 — Максимальный эффект, минимум кода

- [ ] **Lenis — плавный скролл** (`npm install lenis` → `main.jsx`)
- [ ] **Hover на карточках Works** — zoom `scale(1.05)` + тёмный overlay со стрелкой (`Works.css`)
- [ ] **Grain/noise текстура** — `::after` на `body` с SVG-шумом, pure CSS (`App.css`)

## Уровень 2 — Персонажность интерфейса

- [ ] **Кастомный курсор** — белый круг 12px, на hover Works → 60px + «СМОТРЕТЬ» (`Cursor.jsx`)
- [ ] **Параллакс в Hero** — фото движется медленнее страницы (`Hero.jsx`, `Hero.css`)
- [ ] **Счётчик «уже 7 лет»** — цифра отсчитывается от 0 при загрузке (`useCounter.js`)

## Уровень 3 — Редакционный стиль

- [ ] **Анимация заголовка по словам** — каждое слово Hero появляется отдельно снизу вверх
- [ ] **Marquee-строка** между Services и Works — бесконечная прокрутка тегов (`Marquee.jsx`)
- [ ] **Счётчик номеров 01/02/03** в Services при скролле
